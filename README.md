# Flutter E-Commerce App

A modern, responsive, and feature-rich e-commerce application built with Flutter and Dart.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Flutter SDK: [https://flutter.dev/docs/get-started/install](https://flutter.dev/docs/get-started/install)

### Installation

1. Install packages
   ```sh
   flutter pub get
   ```
2. Run the app
   ```sh
   flutter run
   ```
